## 📦 Installation

### 1. Install Python (3.10+)
Windows:  
Download from python.org and enable **Add to PATH**.

Linux:
```bash
sudo apt install python3 -y
```

### 2. Install Git
Windows:  
Download from git-scm.com and enable **Add to PATH**.

Linux:
```bash
sudo apt install git -y
```

### 3. Clone the repository
```bash
git clone https://github.com/YOUR-USERNAME/OSINT-Tool.git
```

### 4. Run the tool
Windows:
```bash
python Main.py
```

Linux:
```bash
python3 Main.py
```
